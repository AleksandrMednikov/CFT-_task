Работа не доделана по пунктам 
-не установлены зависимости в poetry
-нет docker среды для запуска приложения

Полная работа с poetry и docker будет залита на Git Lab. 

Если есть желание проверить работу приложения
1. Используйте .env для настройки переменых. SEC_KEY это время жизни куки для действий требующих авторизаций, полученного благодаря стратегии JWT.
2. Установите библиотеки из requirements.txt
3. Поднимите миграции через алембик 
терминал: alembic upgrade head 
4. Можете запустить тесты
терминал: pytest -v -s tests/  
5. Запустите приложение
терминал: uvicorn main:app --reload 
6. Задйдите на http://127.0.0.1:8000/docs
7. Создайте юзеров через /auth/register, Рекомендую обратить внимание на ввод только email, name, password
8. Установите зарплату и время(необязательный параметр если везде 0) работнику и получите ее за сотрудника. Есть два способа.
	1) Нужно будет вручную установить зарплату на привязаной БД PostgreSQL любому сотруднику в таблице salary
	  Зайдите за этого сотрудника /auth/login
	  Получите его зарплату /salary/get_salary_for_curent_user
	2)Нужно будет вручную установить значени 2 для колоны role_id на привязаной БД PostgreSQL любому сотруднику в таблице user
	  Зайдите за этого сотрудника /auth/login
	  Используйте /salary/insert_salary для ввода зарплаты для сотрудника с указаным user_id
	  Зайдите за сотрудника которому вы установили зарплату /auth/login и получите данные /salary/get_salary_for_curent_user
